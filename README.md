# Hardik Portfolio

Portfolio project tailored for Anton Sten: a product-minded design leader looking for clear thinking, strong judgment, and practical AI-enabled execution.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Project Focus

This is not a generic portfolio. The site should position Hardik as:

- a product-minded builder
- useful in ambiguous design and product work
- strong at prototyping, research synthesis, and workflow design
- capable of technical execution around AI-enabled product work
